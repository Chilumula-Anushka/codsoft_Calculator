# codsoft_Calculator
A simple and responsive Calculator Web Application developed as part of the **CodSoft Web Development Internship**. This calculator performs basic arithmetic operations with a clean and user-friendly interface.

📌 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧹 Clear (AC) button
- ⌫ Delete (Backspace) functionality
- 📱 Responsive design for desktop and mobile
- 🎨 Modern and attractive user interface
